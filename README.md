# 🔐 PhishGuard – URL Phishing Detection Tool

PhishGuard is a **modern cybersecurity web application** designed to detect suspicious or malicious URLs that may be part of phishing attacks.

The tool analyzes a website link using multiple **phishing detection techniques** and provides a **risk score and detailed security report**.

All analysis runs **entirely in the browser**, ensuring **no data is sent to external servers**.

---

# 🧠 Why PhishGuard?

Phishing attacks are one of the **most common cyber threats** today.
Attackers create fake websites that look legitimate to steal:

* Login credentials
* Banking information
* Personal data
* Corporate access credentials

PhishGuard helps users **identify suspicious links before opening them**, reducing the risk of phishing attacks.

# 🔎 Phishing Detection Checks

The tool checks URLs for common phishing indicators:

* Very long URLs
* Excessive special characters (`@`, `_`, `-`, `%`)
* Suspicious keywords such as:

  * login
  * verify
  * secure
  * update
* IP address instead of domain
* Too many subdomains
* Missing HTTPS security

Each factor contributes to a **final phishing risk score**.

---

# 📊 Risk Score System

| Score    | Risk Level    |
| -------- | ------------- |
| 0 – 30   | 🟢 Safe       |
| 31 – 60  | 🟡 Suspicious |
| 61 – 100 | 🔴 High Risk  |

The results are displayed with **visual indicators and a risk meter**.

---

# 📋 Security Report

After analyzing a URL, PhishGuard generates a report including:

* URL length
* HTTPS security status
* Number of suspicious patterns
* Detected phishing keywords
* Number of subdomains
* Overall risk score

This allows users to understand **why a URL may be dangerous**.
